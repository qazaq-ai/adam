# Foundation Scope

## Goal

Build a Kazakh-first text model foundation with clean data, a language-specific
tokenizer, and hard evaluation.

## In Scope

- Kazakh-only corpus policy
- tokenizer research and implementation
- benchmark design
- small-model experimentation
- tiny clean-only deterministic prototype training
- integration with deterministic linguistic validation later

## Out of Scope

- multilingual support
- audio and speech
- multimodal work
- product UI
- cloud orchestration
