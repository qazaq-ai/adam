**Кому**: руководителю учебного / научного / технического подразделения, заинтересованного в казахскоязычной NLP-системе без облачной зависимости.

## Одной строкой

adam — это **казахский школьный помощник на ноутбуке, без облака, без галлюцинаций**, открытый исходник (BUSL-1.1 → Apache-2.0 через 4 года).

## Что есть сейчас (v6.0.0-rc1)

- **Полностью офлайн.** Работает на MacBook Air M2 8 GB или эквиваленте без интернета. Не отправляет данные пользователей наружу.
- **Казахский — родной.** 25 500 корней лексикона, FST-морфология, 144 шаблона диалога. Не «казахский через перевод», а казахский с нуля.
- **3 672 проверенных факта** — география, история, школьные предметы, промышленность РК по областям. Каждый факт ревьюился вручную.
- **Живая дата/время/погода** — adam берёт текущее время из часов ноутбука; погоду через бесплатный Open-Meteo API (только если пользователь явно указал свой город).
- **Голосовой режим** — Whisper STT + macOS TTS. Полностью локально.
- **Прозрачный отказ.** Когда adam не знает ответа — он говорит «не знаю» по-казахски, а не выдумывает.
