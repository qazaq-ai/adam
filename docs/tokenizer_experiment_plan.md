# Tokenizer Experiment Plan

## Objective

Run tokenizer work as repeatable experiments, not ad hoc edits.

## First Experiment

- profile: `adam-kazakh-cyrillic`
- target language: Kazakh
- script: Cyrillic
- goal: improve token efficiency on agglutinative Kazakh text

## Rules

- every tokenizer experiment must have a manifest
- every experiment must point to a declared training manifest
- multilingual fallback experiments are out of scope for the current phase
