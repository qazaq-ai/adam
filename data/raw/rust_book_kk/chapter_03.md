# 3-тарау. Жалпы бағдарламалау ұғымдары

Бұл тарауда `Rust`-та кездесетін негізгі бағдарламалау ұғымдарын
қарастырамыз — олардың көпшілігі басқа тілдерде де кездеседі, бірақ
`Rust`-тың өзіндік мағынасы бар. Айнымалылар, мәлімет түрлері, функциялар,
түсініктемелер және басқару құрылымдары — осы бес ұғымды үйренсеңіз, кез
келген `Rust` бағдарламасын оқи аласыз.

## 3.1. Айнымалылар мен өзгермелілік

`Rust`-та айнымалылар әдепкі бойынша **тұрақты** болады. Бұл — `Rust`-тың
сізге ұсынатын маңызды кепілдіктерінің бірі: егер атау бір рет мәнге
байланса, ол мән кездейсоқ түрде өзгермейді. Сонымен, кодты оқыған кезде
сіз белгілі бір атау тек оған бекітілген мәнді ғана көрсетеді деп сене
аласыз.

Алайда өзгерту қажет болатын жағдайлар бар. Соларға арналған `mut`
кілтсөзін қарастырайық. Жаңа жоба құрып, келесі кодты `src/main.rs`
файлына жазыңыз:

```rust
fn main() {
    let x = 5;
    println!("x мәні: {x}");
    x = 6;
    println!("x мәні: {x}");
}
```

Бағдарламаны құрастырғанда қате аламыз:

```text
error[E0384]: cannot assign twice to immutable variable `x`
 --> src/main.rs:4:5
  |
2 |     let x = 5;
  |         - first assignment to `x`
  |         |
  |         help: consider making this binding mutable: `mut x`
3 |     println!("x мәні: {x}");
4 |     x = 6;
  |     ^^^^^ cannot assign twice to immutable variable
```

Қате анық: тұрақты айнымалыға қайта тағайындау мүмкін емес. Компилятор
сізге қалай түзеуге болатынын ескертіп отыр — `mut` кілтсөзін қосыңыз:

```rust
fn main() {
    let mut x = 5;
    println!("x мәні: {x}");
    x = 6;
    println!("x мәні: {x}");
}
```

Енді бағдарлама дұрыс орындалады және «x мәні: 5», «x мәні: 6» деп
шығарады.

### Тұрақтылар

Тұрақтылар (`const`) — бұл өзгермейтін айнымалыларға ұқсас, бірақ
оларда бірнеше маңызды айырмашылық бар:

- `const` арқылы жарияланған тұрақтыға `mut` қолдануға болмайды — олар
  әрқашан өзгермейді, бұл — `Rust`-тың кепілдігі.
- Тұрақтыны жариялағанда оның түрін **міндетті түрде** көрсету керек:
  `const X: u32 = 100;`. Айнымалымен (`let`) салыстырғанда, түрді
  компилятор қорытындылай алмайды.
- Тұрақтыны кез келген аумақта, оның ішінде ғаламдық аумақта
  жариялауға болады. Бұл бірнеше функцияға ортақ мәнді жасауға ыңғайлы.
- Тұрақтының мәні — компиляция кезінде есептелетін өрнек болуы керек.
  Орындалу кезінде ғана белгілі болатын мән (мысалы, пайдаланушы
  енгізген сан) тұрақты бола алмайды.

`Rust` шарты бойынша тұрақтыларды БАС ҚАРПЕН АТАЙ_БІРЛІГІ_МЕН_БӨЛІП
жазу қабылданған:

```rust
const ҮШ_САҒАТ_ИШИНДЕГІ_МИНУТТАР: u32 = 60 * 60 * 3;
```

Бұл атаулар шартты бекемдікті көрсетеді — кодты оқып шыққан адам
тұрақтыны бірден тани алады.

### Көлеңкелеу

`Rust`-та сіз бір атпен жаңа айнымалы жариялап, бұрынғыны жаба аласыз.
Мұны **көлеңкелеу** (shadowing) деп атайды. Көлеңкелеу — `let` арқылы бір
атпен жаңа байланыстыру жасау. Айнымалыны қайта тағайындау емес — бұл
жаңа айнымалы.

```rust
fn main() {
    let x = 5;

    let x = x + 1;

    {
        let x = x * 2;
        println!("Ішкі аумақтағы x мәні: {x}");
    }

    println!("x мәні: {x}");
}
```

Бұл бағдарлама алдымен `x`-ке 5 мәнін бекітеді. Содан кейін `x = x + 1`
сөйлемі (бірақ `let` арқылы) `x`-тің жаңа айнымалысын жасайды, оның мәні
6. Ішкі фигуралық жақшалардағы аумақта тағы бір `let x = x * 2` жасайды,
бұл аумақта `x` мәні 12. Аумақ біткен соң, ішкі көлеңке жоғалады, ал
сыртқы көлеңке (`x = 6`) қайта көрінеді. Сондықтан бағдарлама былай
шығарады:

```text
Ішкі аумақтағы x мәні: 12
x мәні: 6
```

Көлеңкелеу `mut`-тан өзгеше: `mut`-та біз бар айнымалының мәнін
ауыстырамыз; көлеңкелеуде біз жаңа айнымалы жасаймыз. Айырмашылықтың
маңыздылығы — көлеңкелеу арқылы біз айнымалының *типін* де ауыстыра
аламыз, ал `mut` бұған рұқсат етпейді:

```rust
let spaces = "   ";
let spaces = spaces.len();
```

Бірінші `spaces` — жол, екінші `spaces` — сан. `mut` арқылы бұл мүмкін
емес болар еді, өйткені `mut` түрді сақтайды.

## 3.2. Мәлімет түрлері

`Rust` — статикалық тип жүйесі бар тіл: әрбір мәннің түрі компиляция
кезінде белгілі. Көп жағдайда компилятор түрді қолданысқа қарап өзі
шеше алады, бірақ кейде сізге нақты белгілеу керек болады. Мысалы,
`parse` арқылы жолды санға айналдырғанда:

```rust
let guess: u32 = "42".parse().expect("Сан емес!");
```

Егер `: u32` түр-белгісін алып тастасаңыз, компилятор «қандай түр?» деп
сұрап, қателік шығарады.

`Rust` түрлерін екі топқа бөлуге болады: жалғыз (scalar) және құрама
(compound) типтер.

### Жалғыз типтер

Жалғыз тип бір мәнді көрсетеді. `Rust`-та төрт негізгі жалғыз тип бар:
бүтін сандар, қалқымалы үтірлі сандар, логикалық мәндер және символдар.

#### Бүтін сан түрлері

Бүтін сан түрі — бөлшегі жоқ сандарды сақтайтын түр. `Rust`-та бүтін сан
түрлерінің көп нұсқасы бар:

| Ұзындық | Таңбалы | Таңбасыз |
|---|---|---|
| 8 бит | `i8` | `u8` |
| 16 бит | `i16` | `u16` |
| 32 бит | `i32` | `u32` |
| 64 бит | `i64` | `u64` |
| 128 бит | `i128` | `u128` |
| архитектураға тәуелді | `isize` | `usize` |

Таңбалы түрлер теріс және оң мәндерді қабылдайды; таңбасыздар тек оң
мәндерді (және нөлді). 8 биттік `i8` `-128`-ден `127`-ге дейінгі мәндерді,
ал `u8` `0`-ден `255`-ке дейінгі мәндерді сақтай алады.

`isize` және `usize` түрлерінің ұзындығы машинаның сәулетіне сай:
64 биттік жүйеде 64 бит, 32 биттік жүйеде 32 бит. Бұл түрлерді әдетте
жадтағы индексті немесе ұзындықты сақтау үшін қолданады.

Әдепкі бойынша `Rust` бүтін сан литералын `i32` түрінде қабылдайды.

#### Бүтін санның асып кетуі

Айталық, `u8` түрінде сақталған санға одан да үлкен мәнді (мысалы, 256-ны)
жазсаңыз — бұл *асып кету* (overflow) деп аталады. `Rust` бұған екі
түрлі көзқараспен қарайды:

- **Жөндеу режимінде** (`cargo run`-сіз `--release` жалаушасы) асып кету
  panic-қа ауысады — бағдарлама тоқтайды. Бұл — қателерді ертерек табу
  үшін қолайлы.
- **Шығарылым режимінде** (`cargo run --release`) асып кету **айналу**
  семантикасына көшеді: 256 → 0, 257 → 1 және т.б. Бұл panic-сыз, бірақ
  күтпеген нәтижелер беруі мүмкін, сондықтан асып кетуді нақты ниетпен
  өңдеу керек.

Стандартты кітапхана `wrapping_add`, `checked_add`, `overflowing_add` және
`saturating_add` сияқты әдістерді ұсынады — әрқайсысы асып кетуді басқаша
өңдейді.

#### Қалқымалы үтірлі сандар

Қалқымалы үтірлі сан түрлері — бөлшекті бар сандар. `Rust`-та екеу:
`f32` (32 биттік) және `f64` (64 биттік). Әдепкі бойынша `f64`
қолданылады, өйткені қазіргі CPU-ларда оның жылдамдығы `f32`-ге жуық,
бірақ дәлдігі жоғарырақ.

```rust
fn main() {
    let x = 2.0;      // f64
    let y: f32 = 3.0; // f32
}
```

#### Сандық амалдар

`Rust` стандартты математикалық амалдарды қолдайды: қосу, азайту,
көбейту, бөлу, бөлудің қалдығы:

```rust
fn main() {
    // қосу
    let sum = 5 + 10;

    // азайту
    let difference = 95.5 - 4.3;

    // көбейту
    let product = 4 * 30;

    // бөлу
    let quotient = 56.7 / 32.2;
    let truncated = -5 / 3; // нәтижесі -1; бүтін бөлу қалдықты тастайды

    // қалдық
    let remainder = 43 % 5;
}
```

Бөлгенде `Rust` екі санның түрін шешуге тырысады. Бүтін санды бүтін санға
бөлгенде нәтиже бүтін болады, бөлшегі тасталады.

#### Логикалық тип

Логикалық тип `bool` — екі ғана мәнді қабылдайды: `true` және `false`.
Жадта 1 байт алады. `if` өрнегі мен басқа да басқару құрылымдарының басты
шарты болып табылады.

```rust
fn main() {
    let t = true;
    let f: bool = false;
}
```

#### Символ түрі

`char` түрі бір Unicode символын сақтайды. `Rust`-тың `char`-ы 4 байт
және мүмкіндігі бойынша ASCII-ден көп: эмодзилерді, грек әріптерін,
қазақ кириллицасын — бәрін бір символ ретінде ұстайды.

```rust
fn main() {
    let c = 'z';
    let z: char = 'ㅎ';
    let love = '❤';
}
```

Бір тырнақша (`'`) — символ литералы; қос тырнақша (`"`) — жол литералы.

### Құрама типтер

Құрама тип — бір атпен бірнеше мәнді топтастыратын тип. `Rust`-тың екі
негізгі құрама типі бар: кортеж және жиым.

#### Кортеж типі

Кортеж — әртүрлі типті мәндерді бір топ ретінде ұстайтын тұрақты ұзындықты
құрылым. Кортеждің ұзындығын кейін өзгертуге болмайды.

```rust
fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);
}
```

Кортеждің элементтерін екі тәсілмен алуға болады. Біріншісі —
**бөлшектеу** (destructuring):

```rust
fn main() {
    let tup = (500, 6.4, 1);
    let (x, y, z) = tup;
    println!("y мәні: {y}");
}
```

Бөлшектеу кортеждің әр элементін жеке атаулы айнымалыға бөліп алады.

Екінші тәсіл — нүкте мен индекс арқылы тікелей қол жеткізу:

```rust
fn main() {
    let tup = (500, 6.4, 1);
    let five_hundred = tup.0;
    let six_point_four = tup.1;
    let one = tup.2;
}
```

Кортеждің индексі нөлден басталады.

`()` — арнайы кортеж: ол ешбір элементі жоқ кортеж, *unit* деп аталады.
Көп жағдайда бұл «мән жоқ» дегенді білдіреді — мысалы, өрнегі жоқ
сөйлемнің жасырын қайтару мәні.

#### Жиым типі

Жиым — бір типтегі элементтерден тұратын, тұрақты ұзындықты тізбек.
Кортежден айырмашылығы — барлық элементтер бір типке тиесілі. Жиымның
ұзындығын кейін өзгерту мүмкін емес.

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];
}
```

Жиым стек жадында сақталады. Егер сізге динамикалық ұзындықты ұжым керек
болса, `Vec` түрін қолдану керек (8-тарауда қарастырылады).

Жиымның түрін мен ұзындығын нақты белгілеуге болады:

```rust
fn main() {
    let a: [i32; 5] = [1, 2, 3, 4, 5];
}
```

Барлық элементі бір мәнмен толтырылған жиым жасау үшін:

```rust
fn main() {
    let a = [3; 5]; // [3, 3, 3, 3, 3] — бес элемент, бәрі 3
}
```

Жиым элементтерін индекс арқылы алуға болады:

```rust
fn main() {
    let a = [1, 2, 3, 4, 5];
    let first = a[0];
    let second = a[1];
}
```

Егер сіз жиымның шегінен тыс индексті қолдансаңыз (мысалы, `a[10]`),
`Rust` panic-қа барады. Бұл — `Rust`-тың қауіпсіздік кепілдігі: сіз
тиісті емес жадқа жазып немесе оқимай, қате туралы анық хабарлама
аласыз.

## 3.3. Функциялар

Функция — қайта қолданылатын код блогы. Бір рет жазып, бірнеше жерден
шақыруға болады. `Rust`-та функция `fn` кілтсөзімен жарияланады; әр
бағдарламаның `main` функциясы — кіру нүктесі.

```rust
fn main() {
    println!("Сәлем, дүние!");

    another_function();
}

fn another_function() {
    println!("Басқа функция.");
}
```

`Rust` функция атауларында жалаудай (snake_case) стиль қабылданған —
сөздер арасында астыңғы сызық. Функция кез келген орында, тіпті оны
шақырғаннан кейін де жариялануы мүмкін: `Rust` бағдарламаның бүкіл
кеңістігінен функцияларды табады.

### Параметрлер

Функцияға кіріс мәнін беру үшін параметрлерді қолданамыз:

```rust
fn main() {
    another_function(5);
}

fn another_function(x: i32) {
    println!("x мәні: {x}");
}
```

Параметрдің түрін **міндетті түрде** көрсету керек: `x: i32`. Бұл — `Rust`
дизайнының саналы шешімі: компилятор сізге не дегенді нақты білуі керек,
ал кодты оқыған адам функцияның интерфейсін бір қарап-ақ түсінсе болады.

Бір функцияда бірнеше параметр болуы мүмкін — әрбірі қос нүктемен
бөлінеді:

```rust
fn main() {
    print_labeled_measurement(5, 'h');
}

fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("Өлшем: {value}{unit_label}");
}
```

### Сөйлемдер мен өрнектер

`Rust`-та өрнек пен сөйлемнің айырмашылығы маңызды. **Сөйлем** — әрекет
орындайтын бірлік, ол мән қайтармайды. **Өрнек** — мән өндіретін бірлік.

Мысалы:

```rust
fn main() {
    let y = 6;
}
```

`let y = 6;` — сөйлем. Ол `y`-ке 6-ны бекітеді, бірақ ол мән қайтармайды;
сондықтан `let x = (let y = 6);` сияқты жазуға болмайды.

Өрнек кодтың көп бөлігін құрайды. Жай шақыру (`5 + 6`), функция шақыруы
(`add(2, 3)`), макрос шақыруы (`println!()`), фигуралық жақшамен оралған
блок — бәрі өрнек:

```rust
fn main() {
    let y = {
        let x = 3;
        x + 1
    };
    println!("y мәні: {y}");
}
```

Ішкі блоктың мәні — соңғы өрнек, `x + 1`. Маңызды: бұл өрнек **нүктелі
үтірмен аяқталмайды**. Егер нүктелі үтір қоятын болсаңыз, ол сөйлемге
айналады және мән қайтармайды. Сондықтан блок өзінің ішіндегі ең соңғы
өрнектің мәніне айналады.

### Мән қайтаратын функциялар

Функция мән қайтара алады. Қайтару типін `->` арқылы көрсетесіз:

```rust
fn five() -> i32 {
    5
}

fn main() {
    let x = five();
    println!("x мәні: {x}");
}
```

`five` функциясы `i32` қайтарады. Денесі — бір ғана өрнек: `5`. Ол нүктелі
үтірсіз, сондықтан өрнек ретінде функцияның қайтарар мәніне айналады.

`return` кілтсөзін де қолдана аламыз — әсіресе функцияны ертерек
шығарғанда:

```rust
fn plus_one(x: i32) -> i32 {
    x + 1
}
```

`x + 1` соңғы өрнек, нүктелі үтірсіз, сондықтан ол қайтарылады. Егер
нүктелі үтір қойсаңыз — қате аласыз: «expected `i32`, found `()`».

## 3.4. Түсініктемелер

Кодқа түсініктемелер қосу — оқу мен ой бөлісу үшін маңызды. `Rust`-та
түсініктемелер `//` арқылы басталады және жол соңына дейін созылады:

```rust
// Бұл — түсініктеме.
fn main() {
    // `Rust` бағдарламасының кіру нүктесі.
    println!("Сәлем!");
}
```

Көп жолды түсініктемені әдетте әр жолды жеке `//` арқылы жазу қабылданған:

```rust
// Бірінші жол түсініктемесі.
// Екінші жол.
// Үшінші жол.
```

`Rust` `/* ... */` стиліндегі блок түсініктемелерін де қабылдайды, бірақ
жолдық `//` стилі әлдеқайда жиі қолданылады.

Сонымен қатар `Rust`-та *құжаттама түсініктемесі* (`///`) бар — ол сыртқы
кітапханаларға арналған құжаттаманы автоматты түрде шығаруға мүмкіндік
береді. 14-тарауда олар туралы толығырақ айтамыз.

## 3.5. Басқару құрылымдары

Шартты тармақтау мен цикл — бағдарламалаудың негізгі құралдары. `Rust`-та
екі негізгі басқару құрылымы бар: `if` өрнегі және циклдер.

### `if` өрнегі

`if` өрнегі шартты тексеріп, тармақ таңдайды. Шарт `bool` түрінде болуы
керек — `Rust` басқа тілдерден өзгешелігі сонда: «нөл — жалған, нөлсіз
— шын» сияқты автоматты түрлендіру жасамайды.

```rust
fn main() {
    let number = 3;

    if number < 5 {
        println!("Шарт ақиқат болды.");
    } else {
        println!("Шарт ақиқат болмады.");
    }
}
```

`else if` арқылы бірнеше шартты тізбектеуге болады:

```rust
fn main() {
    let number = 6;

    if number % 4 == 0 {
        println!("4-ке бөлінеді.");
    } else if number % 3 == 0 {
        println!("3-ке бөлінеді.");
    } else if number % 2 == 0 {
        println!("2-ге бөлінеді.");
    } else {
        println!("4, 3 немесе 2-ге бөлінбейді.");
    }
}
```

Бірнеше `else if` блогы кодты жайыр етеді. Көбіне `match` өрнегін
қолдану ыңғайлырақ — 6-тарауда қарастырылады.

### `if` өрнегін `let` сөйлеміне қосу

`if` — өрнек, сондықтан оның нәтижесін айнымалыға бекітуге болады:

```rust
fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("number мәні: {number}");
}
```

Маңызды: `if` мен `else` тармақтарының түрлері бірдей болуы керек.
Жоғарыдағы мысалда екі тармақ та `i32` мәнін қайтарады. Егер сіз бір
тармақта сан, екіншісінде жол қайтарсаңыз, компилятор қателікке барады.

### Циклдар

`Rust` үш цикл құрылымын ұсынады: `loop`, `while`, `for`.

#### `loop` — шарт жоқ цикл

`loop` денесін шарт қойылғанша қайталай береді. Циклдан тек `break`
арқылы шығуға болады:

```rust
fn main() {
    loop {
        println!("Қайтадан!");
    }
}
```

Бұл бағдарлама мәңгілікке «Қайтадан!» деп шығарып отырады. `Ctrl-C`
арқылы тоқтатуға тура келеді.

#### Циклдан мән қайтару

`break` сөзінен кейін өрнек көрсетсеңіз — ол мән цикл өрнегінің мәніне
айналады:

```rust
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;
        }
    };

    println!("Нәтиже: {result}");
}
```

Цикл `counter` 10-ға жеткенде `counter * 2 = 20`-ны қайтарып шығады. Сол
мән `result`-ке бекітіледі.

#### Цикл белгілері

Бір циклдан екіншісіне `break` арқылы шығу — ыңғайсыз. Сондықтан `Rust`
**цикл белгілерін** ұсынады. Белгі апостроф пен атаудан тұрады. `break`
немесе `continue` нақты қай циклді көздегеніңізді осы белгімен
көрсетесіз:

```rust
fn main() {
    let mut count = 0;
    'counting_up: loop {
        let mut remaining = 10;

        loop {
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up;
            }
            remaining -= 1;
        }

        count += 1;
    }
    println!("count = {count}");
}
```

Сыртқы цикл `'counting_up` деген белгімен жарияланған. `break 'counting_up`
ішкі циклден шығып, бірден сыртқы циклді де үзеді.

#### `while` цикл

`while` цикл шарт ақиқат болғанда денесін қайталайды:

```rust
fn main() {
    let mut number = 3;

    while number != 0 {
        println!("{number}!");
        number -= 1;
    }

    println!("Аттан!");
}
```

`while` — `loop` мен `if`-тың қосындысы. Бар санды нөлге дейін кеміту
немесе шарт орындалғанша күту — `while`-тың әдеттегі қолданысы.

#### `for` цикл

Жиым немесе диапазон бойынша жүруге `for` цикл ыңғайлы. Ол `Rust`-та
ең жиі қолданылатын цикл түрі.

```rust
fn main() {
    let a = [10, 20, 30, 40, 50];

    for element in a {
        println!("{element}");
    }
}
```

`for` циклы итераторды аралайды. Бұл `while` арқылы жасағанға қарағанда
қауіпсіз: жиым ұзындығы өзгерсе де, индекс шегінен шығып кету
мүмкіндігі жоқ.

Тек белгілі санға дейінгі сандарды кезу үшін **диапазон өрнегін**
қолдануға болады:

```rust
fn main() {
    for number in 1..4 {
        println!("{number}!");
    }
    println!("Аттан!");
}
```

`1..4` — қамтылмаған диапазон: 1, 2, 3 (4 кірмейді). `1..=4` — қамтылған
диапазон: 1, 2, 3, 4. `.rev()` әдісі диапазонды кері тәртіппен жүреді:

```rust
for number in (1..4).rev() {
    println!("{number}!");
}
```

Бұл «3!», «2!», «1!», «Аттан!» деп шығарады.

## 3.6. Тарау қорытындысы

Бұл тарауда `Rust`-тың негізгі ұғымдарын қарастырдық: айнымалылар мен
тұрақтылар, өзгермелілік, көлеңкелеу, мәлімет түрлері (бүтін сандар,
қалқымалы сандар, символдар, кортеждер, жиымдар), функциялар (параметрлер,
қайтару мәндері, өрнек пен сөйлем айырмашылығы), түсініктемелер, басқару
құрылымдары (`if`, `loop`, `while`, `for`).

Келесі тарауда `Rust`-тың ең ерекше және ең күшті ұғымына — иелік
жүйесіне көшеміз. Иелік модельі дегеніміз — `Rust`-тың жадыны қалай
қауіпсіз басқаратынының негізі, бұл басқа жүйелік тілдерде кездеспейтін
тұжырымдама.
