# 1-тарау. Бастау

Осы тарауда `Rust` тілімен жұмыс істеуді қалай бастау керектігін көрсетеді.
Алдымен қажетті құралдарды орнатамыз, содан кейін бірінші бағдарламаны жазып,
оны Cargo көмегімен басқаруды үйренеміз. Тарау үш бөлімнен тұрады:

1. **Орнату** — `rustup` арқылы `Rust` құрал жинағын орнату.
2. **Hello, World!** — алғашқы `Rust` бағдарламасын жазу және орындау.
3. **Hello, Cargo!** — Cargo көмегімен жобаны жасау, құрастыру және басқару.

Тараудың соңында сіз өз компьютеріңізде `Rust` бағдарламасын жаза алатын
және оны орындай алатын боласыз.

## 1.1. Орнату

### `rustup` дегеніміз не

`Rust`-ты орнатудың ең сенімді жолы — `rustup` атты ресми құрал арқылы орнату.
`rustup` — `Rust` нұсқаларын басқаратын құрал. Ол сізге қажетті нұсқаны орнатады,
оны жаңартады және әртүрлі құрастыру мақсаттарына арналған құрал жинақтарын
қосады.

`rustup` арқылы орнатқанда, сіздің машинаңызға бірнеше зат қатар келеді:
`rustc` компиляторы, `cargo` құрылыс құралы, `rustfmt` пішімдеуші және
бастапқы Rust стандартты кітапханасы.

### Linux немесе macOS жүйесінде орнату

Егер сіз Linux немесе macOS жүйесінде жұмыс істейтін болсаңыз, терминалды ашып,
келесі команданы енгізіңіз:

```sh
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Бұл команда `rustup` орнатушы сценарийін жүктеп алады және оны орындайды.
Сценарий сізге орнату нұсқаларын ұсынады; әдетте әдепкі нұсқа жеткілікті,
сондықтан `1` пернесін басып немесе `Enter` пернесін басып жалғастырыңыз.

Орнату аяқталғаннан кейін терминалды қайта іске қосыңыз. Бұл `Rust` командаларын
жүйелік `PATH` айнымалысына қосу үшін қажет.

`Rust`-тың сәтті орнатылғанын тексеру үшін терминалда келесі команданы енгізіңіз:

```sh
$ rustc --version
```

Сіз қазіргі нұсқаны, коммит идентификаторын және коммит күнін көресіз. Мысалы:

```text
rustc 1.75.0 (82e1608df 2023-12-21)
```

### Windows жүйесінде орнату

Windows жүйесінде https://www.rust-lang.org/tools/install сайтындағы нұсқауларға
сәйкес `rustup-init.exe` файлын жүктеп алып, оны іске қосыңыз. Орнатушы сізге
қандай орнату түрін қолданатыныңызды сұрайды; әдепкі нұсқаны таңдау жеткілікті.

Windows-та `Rust`-ты құрастыру үшін `MSVC build tools` орнатылған болуы керек.
Орнатушы сізге қажет болса оны алдын ала жүктеп алуды ұсынады.

### Жаңарту және өшіру

`Rust`-ты жаңарту үшін терминалда мынаны енгізіңіз:

```sh
$ rustup update
```

Өшіру үшін:

```sh
$ rustup self uninstall
```

### Жергілікті құжаттама

`Rust`-ты орнатқан кезде құжаттаманың жергілікті көшірмесі де орнатылады, оны
интернетке шықпай-ақ оқуға болады. Терминалда `rustup doc` командасын енгізсеңіз,
шолғышта стандартты кітапхана құжаттамасы ашылады.

## 1.2. Hello, World!

`Rust`-ты орнатқан соң, бірінші бағдарламаны жазайық. Дәстүр бойынша жаңа тілмен
танысқанда «Hello, world!» сөзін экранға шығаратын кішкене бағдарламадан бастау
қабылданған. Біз де солай етеміз.

### Жоба қалтасын жасау

Алдымен `Rust` кодыңызды сақтайтын қалта жасаңыз. `Rust`-қа кодтың қайда тұрғаны
маңызды емес, бірақ біз бұл кітаптағы жаттығулар үшін `projects` атты қалта
жасап, ішінде барлық жобаларды сақтауды ұсынамыз.

Терминалда келесі командаларды енгізіңіз:

```sh
$ mkdir ~/projects
$ cd ~/projects
$ mkdir hello_world
$ cd hello_world
```

Windows-та `cmd` ішінде:

```cmd
> mkdir "%USERPROFILE%\projects"
> cd /d "%USERPROFILE%\projects"
> mkdir hello_world
> cd hello_world
```

### `Rust` бағдарламасын жазу және орындау

Жаңа бастапқы файл жасап, оны `main.rs` деп атаңыз. `Rust` файлдарының
кеңейтімі әрқашан `.rs` болады. Егер файл атауында бірнеше сөз болса, оларды
астыңғы сызық арқылы бөлу қабылданған: мысалы, `hello_world.rs` (бірақ
`helloworld.rs` емес).

`main.rs` файлын ашып, ішіне мына кодты жазыңыз:

```rust
fn main() {
    println!("Hello, world!");
}
```

Файлды сақтап, терминалға қайтыңыз. Linux немесе macOS жүйесінде келесі екі
команданы енгізіңіз:

```sh
$ rustc main.rs
$ ./main
Hello, world!
```

Windows-та `./main` орнына `.\main.exe` жазыңыз. Қай жүйеде болса да, экранда
`Hello, world!` сөзі шығуы керек. Егер шықпаса, орнату қадамына қайтып, барлығы
дұрыс орындалғанын тексеріңіз.

### `Rust` бағдарламасының құрылымы

Жаңа жазған бағдарламаны жолма-жол қарастырайық. Бірінші жолда:

```rust
fn main() {
```

Бұл жол `main` атты функцияны жариялайды. `main` функциясы — кез келген
орындалатын `Rust` бағдарламасында ең бірінші іске қосылатын код. Ашатын
дөңгелек жақшалардың ішінде параметрлер болады; бұл жағдайда олар жоқ.
Әрі қарай, функция денесі фигуралық жақшалармен оралады.

Функция денесінде бір ғана сөйлем бар:

```rust
    println!("Hello, world!");
```

Бұл сөйлем «Hello, world!» сөзін экранға шығарады. Назар аударатын төрт жайт:

1. `Rust` стилі төрт бос орынмен шегініс жасайды, ал tabulation символын
   қолданбайды.
2. `println!` — бұл функция емес, **макрос**. Леп белгісі (`!`) макросты
   функциядан ажыратып тұр. Макростар туралы 19-тарауда айтылады.
3. Жолды дөңгелек жақшалардың ішінде «Hello, world!» жолын беріп шақырамыз —
   бұл аргумент.
4. Сөйлем нүктелі үтірмен (`;`) аяқталады. Нүктелі үтір сөйлемнің біткенін
   білдіреді және келесісі басталуы мүмкін екенін көрсетеді. `Rust` кодының
   көп сөйлемі нүктелі үтірмен аяқталады.

### Құрастыру мен орындау — бұл екі бөлек қадам

`Rust` — алдын ала құрастырылатын тіл. Бағдарламаны орындамас бұрын оны
`rustc` компиляторы арқылы орындалатын файлға айналдыру керек. Біз дәл осыны
жоғарыда жасадық: `rustc main.rs` командасы `main.rs` файлынан `main` (немесе
Windows-та `main.exe`) деген орындалатын файл жасады. Содан кейін біз сол
файлды тікелей орындай алдық.

Бұл интерпретатор тілдерінен (мысалы Python немесе JavaScript) өзгеше: онда
сіз кодты жазып, `python script.py` деп тікелей орындайсыз. `Rust`-та екі
қадам бөлек: алдымен құрастыру, содан кейін орындау.

Бір рет құрастырылған орындалатын файлды бастапқы кодсыз да басқа компьютерге
жіберіп орындай беруге болады. Бұл — алдын ала құрастырылатын тілдердің
артықшылықтарының бірі.

Бірақ `rustc` арқылы тікелей құрастыру тек қарапайым бағдарламалар үшін
ыңғайлы. Жоба үлкейген сайын тәуелділіктер пайда болады, оларды басқару
қиындай түседі. Сондықтан келесі бөлімде Cargo туралы айтамыз — ол тура осы
мақсатқа арналған құрал.

## 1.3. Hello, Cargo!

Cargo — `Rust`-тың құрылыс жүйесі әрі бума менеджері. Cargo сіздің
кодыңызды құрастырады, оның тәуелділіктерін жүктеп алады, тестілерді іске
қосады және басқа да көмекші тапсырмаларды орындайды. `Rust` қауымдастығы
жобаларының басым көпшілігі Cargo арқылы басқарылады, сондықтан бұл бөлімде
оны қолдануды үйренеміз.

Cargo `rustup` арқылы `Rust`-пен бірге орнатылады. Cargo орнатылғанын тексеру
үшін терминалда келесі команданы енгізіңіз:

```sh
$ cargo --version
```

### Cargo көмегімен жоба жасау

Жаңа жобаны Cargo арқылы жасайық. `projects` қалтасына өтіп, мынаны енгізіңіз:

```sh
$ cargo new hello_cargo
$ cd hello_cargo
```

`cargo new hello_cargo` командасы `hello_cargo` атты жаңа қалта жасайды
және оның ішінде жобаға қажетті құрылымды дайындайды. Қалтаны ашып қарасаңыз,
Cargo сізге үшеуін дайындағанын көресіз: `Cargo.toml` файлы, `src` атты
қалта және оның ішінде `main.rs` файлы. Cargo сонымен қатар `git` жобасын
бастайды (егер `hello_cargo` бұған дейін `git` қоймасында болмаса).

`Cargo.toml` файлын мәтін редакторымен ашыңыз. Ішінде мынадай нәрсе болады:

```toml
[package]
name = "hello_cargo"
version = "0.1.0"
edition = "2021"

[dependencies]
```

Бұл файл — TOML пішіміндегі баптау файлы. Алғашқы бөлік — `[package]` —
жобаның атауын, нұсқасын және `Rust`-тың қай басылымын қолданатынын
сипаттайды. Екінші бөлік — `[dependencies]` — жобаның тәуелділіктерін
тізбелейтін бөлік. `Rust`-та тәуелділіктер «сандық» (`crate`) деп аталады;
бұл жобада әзірге тәуелділік жоқ.

Енді `src/main.rs` файлын ашыңыз. Cargo сізге дайын `Hello, world!`
бағдарламасын жасап қойған:

```rust
fn main() {
    println!("Hello, world!");
}
```

Cargo жобаның бастапқы кодын `src` қалтасына сақтайтыны шартты. Жобаның
түбірлік қалтасы тек `Cargo.toml`, `README.md`, лицензия және басқа да
конфигурация файлдары үшін.

### Cargo жобасын құрастыру және орындау

`hello_cargo` қалтасында тұрып келесіні енгізіңіз:

```sh
$ cargo build
   Compiling hello_cargo v0.1.0 (file:///projects/hello_cargo)
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 2.85 secs
```

Бұл команда `target/debug/hello_cargo` (немесе Windows-та
`target\debug\hello_cargo.exe`) деген орындалатын файлды жасайды. Әдепкі
бойынша Cargo `dev` (даму) бейіні арқылы құрастырады — бұл құрастыру жылдам,
бірақ кодтың орындалу жылдамдығы жоғары емес.

Орындалатын файлды іске қосу үшін:

```sh
$ ./target/debug/hello_cargo
Hello, world!
```

Бірақ әдетте жоғарыдағы екі қадамды бір команда арқылы орындау ыңғайлы:

```sh
$ cargo run
   Compiling hello_cargo v0.1.0 (file:///projects/hello_cargo)
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.38 secs
     Running `target/debug/hello_cargo`
Hello, world!
```

`cargo run` командасы алдымен жобаны құрастырады (өзгеріс болса), содан кейін
орындайды. Әзірлеуші ретінде сіз бұл команданы ең жиі қолданасыз.

Тағы пайдалы команда — `cargo check`:

```sh
$ cargo check
   Compiling hello_cargo v0.1.0 (file:///projects/hello_cargo)
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.32 secs
```

`cargo check` кодтың құрастырылатынын тексереді, бірақ нақты орындалатын
файл жасамайды. Сондықтан ол `cargo build`-тен әлдеқайда жылдам жұмыс
істейді. Әзірлеу барысында кодтың дұрыс жазылғанын жиі тексеріп отыруға
ыңғайлы.

### Шығарылым үшін құрастыру

Бағдарламаны соңғы пайдаланушыға берерде немесе оның жылдамдығын өлшегенде
оны `cargo build --release` арқылы құрастырыңыз:

```sh
$ cargo build --release
```

Бұл команда оптимизациялары қосылған, орындалу жылдамдығы жоғары орындалатын
файлды жасайды; нәтиже `target/release` қалтасына түседі. Оптимизация
құрастыру уақытын ұзартады, бірақ бағдарламаның орындалу жылдамдығын едәуір
арттырады. Сондықтан Cargo-ның екі бейіні бар: даму бейіні (жылдам құрастыру,
жөндеу үшін) және шығарылым бейіні (баяу құрастыру, жылдам орындау).

### Cargo — шартты келісім

Қарапайым жобалар үшін Cargo `rustc` арқылы тікелей құрастырумен салыстырғанда
үлкен артықшылық бермейді. Бірақ жоба үлкейген сайын Cargo-ның пайдасы
анығырақ көрінеді: тәуелділіктерді қосу, сынақтарды іске қосу, бірнеше
сандықтан тұратын кітапхананы басқару — бәрі Cargo арқылы оңай жасалады.

`Rust` қауымдастығының жобалары `cargo new` арқылы жасалатын құрылымды
ұстанады. Сондықтан кез келген ашық кодты `Rust` жобасын жүктеп алып, оны:

```sh
$ git clone <url>
$ cd <project>
$ cargo build
```

командаларымен құрастыра аласыз.

### Тарау қорытындысы

Осы тарауда сіз `Rust`-ты орнаттыңыз, бірінші бағдарламаңызды жаздыңыз және
Cargo көмегімен жобаны қалай басқаруды үйрендіңіз. Келесі тарауда нағыз
жаттығу — санды табу ойыны — арқылы тілдің негізгі ұғымдарымен жақын
танысамыз: айнымалылар, мәлімет түрлері, басқару құрылымдары, сыртқы
сандықтарды қосу.
